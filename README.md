# Banking Account Management System (BAMS)

A modern, secure, and professional digital banking platform designed to automate and streamline banking operations. BAMS allows customers to manage their finances from anywhere with ease and security.

## 🚀 Features

### Public Features
- **Modern Homepage**: Interactive UI with statistics and services overview.
- **Product Portal**: Detailed information on Savings, Loans, and Credit Cards.
- **About Us**: Bank history, mission, and leadership team.
- **Contact & Support**: Branch locator and interactive support forms.
- **Help Center**: Searchable FAQ and help ticket system.

### Banking Operations
- **Secure Authentication**: Multi-factor authentication and role-based access.
- **Account Management**: Open and manage multiple account types.
- **Transaction System**: Seamless deposits, withdrawals, and transfers.
- **Dashboard**: Real-time analytics and transaction history.

## 🛠 Technology Stack

- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+).
- **Backend**: Java (Spring Boot - Planned).
- **Database**: SQL (MySQL/PostgreSQL - Planned).
- **Styling**: Custom Design System with Inter & Poppins fonts.

## 📂 Project Structure

- `frontend/`: Public website and dashboard UI.
- `backend/`: Core banking logic and API services.
- `database/`: SQL scripts and schema definitions.
- `docs/`: Documentation and project reports.
- `assets/`: Static assets and branding material.
- `screenshots/`: Project visuals.
